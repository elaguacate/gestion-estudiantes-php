## Objetivo

Desarrollar una aplicación simple en **PHP** que aplique los conceptos de la **Programación Orientada a Objetos (POO)**:
- Creación de clases y definición de atributos  
- Instanciación de objetos  
- Implementación de métodos y constructores  
- Aplicación de herencia, encapsulamiento y polimorfismo  

---

## Tecnologías utilizadas

- Lenguaje: **PHP**  
- Entorno: **OnlineGDB**  
- Control de versiones: **GitHub**

---

## Descripción general del proyecto

El proyecto simula un pequeño sistema de **gestión de estudiantes**, donde se crean objetos de tipo `Estudiante` que heredan de la clase `Persona`.  
Cada estudiante puede tener calificaciones, y el programa calcula su promedio, muestra información detallada y controla errores como matrículas duplicadas.
